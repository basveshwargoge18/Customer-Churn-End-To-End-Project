# 📊 Customer Churn Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the Customer Churn dataset to understand customer behavior and identify the key factors contributing to customer churn. Through statistical analysis and data visualization, the project uncovers valuable insights that can help businesses improve customer retention and reduce churn.

---

## 🎯 Objectives

- Understand the structure and characteristics of the dataset.
- Identify and handle missing values and duplicate records.
- Detect and analyze outliers.
- Perform univariate and bivariate analysis.
- Explore relationships between variables and customer churn.
- Visualize important patterns and trends in the data.
- Extract meaningful business insights from the dataset.

## 💡 Key Insights

- Customers with an international plan tend to churn more frequently.
- Higher customer service calls are associated with increased churn.
- Certain customer usage patterns significantly influence customer retention.
- Correlation analysis helps identify important features affecting churn.
- Data visualization reveals hidden trends and relationships within the dataset.

## 📊 Business Insights
