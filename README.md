# 📊 Customer Churn Analysis Using Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on a Customer Churn dataset to understand customer behavior, identify patterns, and extract business insights that can help reduce customer attrition and improve retention strategies.

The project covers data cleaning, missing value analysis, duplicate handling, outlier treatment, univariate analysis, bivariate analysis, and business-driven insights.

---

## 🎯 Objectives

- Import and explore the customer churn dataset.
- Understand the structure and characteristics of the data.
- Perform data cleaning and preprocessing.
- Detect and handle missing values.
- Identify and remove duplicate records.
- Detect and treat outliers.
- Perform Exploratory Data Analysis (EDA).
- Analyze relationships between features and churn.
- Extract meaningful business insights.
- Support data-driven decision-making.

---
